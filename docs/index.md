# 歡迎來到川寶每日報告

歡迎使用川寶團隊每日報告自動發布平台！🎉

## 關於本站

本站用於自動發布川寶團隊的每日報告，包括：

- 📊 **投資分析報告** - 每日市場分析與投資建議
- 📰 **產業研究報告** - 各產業深度研究與趨勢分析
- 💼 **財務報告** - 財務狀況與營運表現追蹤

## 最新報告

請至 [報告列表](./reports/index.md) 查看所有報告。

## 使用說明

### 自動發布
每日報告會自動部署到 Vercel，無需手動操作。

### 手動部署
如需手動發布，請執行：

```bash
cd ~/.openclaw/workspace/reports_site
./deploy_to_vercel.sh
```
